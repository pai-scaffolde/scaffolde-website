# Scaffolde Website Visual Asset Prompts

Generated after visual QA failure: the prior redesign was too text-heavy and had zero images.

## Shared art direction
Original editorial/technical illustration system for Scaffolde.ai. Not generic SaaS. No stock people, no purple gradients, no 3-column icon grid. Warm off-white paper background, charcoal ink, signal orange accent (#D9621C), blueprint cyan micro-lines, layered systems diagrams, agent traces, visible verification/proof marks. Should feel like a serious AI operating system, a capability factory, and a lab notebook. High contrast, crisp vector-meets-editorial look, suitable for web hero and section imagery.

## 01 Capability Factory Hero
Landscape 16:9 hero artwork. Show a layered AI capability factory: modular workcells labeled conceptually by shape not text, agent lanes branching from an intent core, memory stacks, verification gates, browser/repo/chat/document surfaces, and a proof ledger. Composition should have strong depth and focal point. No readable fake text. Warm off-white canvas, charcoal structures, orange highlights, cyan trace lines. Premium technical editorial illustration.

## 02 Operating Loop Diagram
Square diagram artwork. A five-stage operating loop: define, dispatch, coordinate, verify, learn, represented visually as orbiting stations around a central artifact. Include small checkmarks/probe marks and arrows, memory fragments feeding back into skills. No readable fake text. Same Scaffolde palette: off-white, charcoal, orange, subtle cyan.

## 03 Humans + Agents Surface Map
Landscape 16:9 artwork. Split-but-connected composition: left side human-facing website/manual/proof surfaces, right side agent-facing llms.txt/json/markdown/API routes, bridged by Scaffolde orchestration. Show browser windows, markdown sheets, JSON blocks, terminal panes as abstract readable surfaces without fake text. Same palette, crisp editorial system diagram, engaging not generic.
